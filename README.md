# ポケモン当てクイズアプリ

ヒントを見てポケモン名を当てる、単一HTMLのクイズアプリです。

## 特徴

- 単一HTMLで動作
- 外部通信なし
- 外部ライブラリなし
- スマホ対応
- 4択モード / 入力モード
- 難易度切替
- 地方別フィルター
- ヒント段階表示
- 最高スコア保存

## 使い方

`index.html` をブラウザで開くだけです。

## GitHub Pagesで公開する場合

1. GitHubでこのリポジトリを開く
2. Settings
3. Pages
4. Sourceを `Deploy from a branch`
5. Branchを `main`、フォルダを `/root` に設定
6. Save

数分後に以下のURLで開けます。

```text
https://kouheim1979.github.io/pokemon-quiz-app/
```

このアプリは学習・クイズ用の非公式ファン制作サンプルです。公式画像・音声・外部データは使用していません。
